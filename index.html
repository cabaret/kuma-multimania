Adobe Edge Tools with a "Bang"

(Serge Jespers)


  • The new Edge tools look nice...
  • but this was an Adobe product pitch.

  • Hence, some critical remarks.

  • (All Edge tools are still free on CC for the time being)

Edge Animate


  • Resembles Flash, for canvas animations
  • Looks okay for quick and dirty mockups
  • Guy seemed reluctant to show code..
  • When he did, it turned out to be a mess

  • Decent tool for designers to let us know how
    they want things exactly

Edge Reflow


  • Easily create mockups for responsive web design
  • Not quite sure how this works design-wise

  • Awesome tool to see how a design should react
    to various breakpoints

Edge Code


  • Code Editor
  • Built with HTML, CSS, JS: easily configurable/adaptable
  • Edge Inspect, Edge Webfonts, Phonegap Build integration

  • I personally don't see how this beats
    Sublime + package ecosystem
